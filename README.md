# ESP POMODORO

Pomodoro timer built on ESP32 chip

# Requirements

- ESP-IDF setup
- Create / generate `sdkconfig` file for your specific chip, for example:

```sh
idf.py set-target esp32c6
```

# Hardware

- Passif buzzer for sound (optional)
- GC9A01 SPI display, i'm using a cheap IPS 1.28inch round display from aliexpress
